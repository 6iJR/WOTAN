Die Fakultät ist eine Funktion, die einer natürlichen Zahl das Produkt aller natürlichen Zahlen (ohne Null)
kleiner und gleich dieser Zahl zuordnet. Sie soll mithilfe von Rekursion implementiert werden.
Eine rekursive Methode ist eine Methode, die sich selbst aufruft.

Das Schreiben von Klassenkopf sowie main-Methode ist nicht nötig, es soll nur der Methodenrumpf ergänzt werden.

public int factorial(int n) {
  if (n == 1) return 1;
  
  return n * factorial(n-1);
}
