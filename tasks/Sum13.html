Die Summe aller im Array nums befindlichen Zahlen soll zurückgegeben werden.
Die Zahl 13 ist aber eine Unglückszahl, diese zählt nicht, das Gleiche gilt
auch für die Zahl, die direkt hinter der 13 ist.

Das Schreiben von Klassenkopf sowie main-Methode ist nicht nötig, es soll nur der Methodenrumpf ergänzt werden.

public int sum13(int[] nums) {
  int sum = 0;
  for(int i = 0; i < nums.length; i++) {
    if(nums[i] != 13) {
      sum += nums[i];
    } else {
      i++;
    }
  }
  return sum;
}
